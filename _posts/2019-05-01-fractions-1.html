Convert each of the improper fractions below into a decimal to nearest hundreth, a mixed fraction and find an equivalent fraction that has a denominator of 24.

  1. $$\fraction{15}{6}$$
  2. $$\fraction{7}{2}$$
  3. $$\fraction{100}{48}$$
  4. $$\fraction{48}{36}$$
{% capture solution %}
  1. Decimal: $$\fraction{15}{6} = 2.50$$ Mixed fraction: $$\fraction{15}{6} = 2\fraction{3}{6} = 2\fraction{1}{2}$$ Equivalent fraction: $$\fraction{15}{6} = \fraction{60}{24}$$
  2. Decimal: $$\fraction{7}{2} = 3.50$$ Mixed fractions: $$\fraction{7}{2} = 3\fraction{1}{2}$$ Equivalent fraction: $$\fraction{7}{2} = \fraction{84}{24}$$
  3. Decimal: $$\fraction{100}{48} = 2.08$$ Mixed fraction: $$\fraction{100}{48} = 2\fraction{4}{48} = 2\fraction{1}{12}$$ Equivalent fraction: $$\fraction{100}{48} = \fraction{50}{24}$$
  4. Decimal: $$\fraction{48}{36} \approx 1.33$$ Mixed fraction: $$\fraction{48}{36} = 1\fraction{12}{36} = 1\fraction{1}{3}$$ Equivalen fraction: $$\fraction{48}{36} = \fraction{32}{24}$$
