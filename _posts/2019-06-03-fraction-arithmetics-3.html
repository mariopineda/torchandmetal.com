Evaluate without a calculator.

  1. $$\fraction{1}{6} \div \fraction{8}{11}$$
  2. $$\fraction{1}{3} \div \fraction{13}{9}$$
  3. $$\fraction{3}{2} \div \fraction{9}{4}$$
  4. $$\fraction{9}{5} \div \fraction{20}{3}$$
{% capture solution %}
  1. $$\fraction{1}{6} \div \fraction{8}{11} = \fraction{1}{6} \times \fraction{11}{8} = \fraction{11}{48}$$
  2. $$\fraction{1}{3} \div \fraction{13}{9} = \fraction{1}{3} \times \fraction{9}{13} = \fraction{1}{1} \times \fraction{3}{13} = \fraction{3}{13}$$
  3. $$\fraction{3}{2} \div \fraction{9}{4} = \fraction{3}{2} \times \fraction{4}{9} = \fraction{1}{1} \times \fraction{2}{3} = \fraction{2}{3}$$
  4. $$\fraction{9}{5} \div \fraction{20}{3} = \fraction{9}{5} \times \fraction{3}{20} = \fraction{27}{100}$$
