Without using a calculator, determine the following characteristics for the following exponential functions,

  • y-intercept
  • End Behaviour
  • Increasing or decreasing exponential function
  • Common Ratio
  • Domain
  • Range
  • x-intercept
  1. $$y=2^x$$
  2. $$y=2(10)^x$$
  3. $$y= \left( \fraction{1}{2}\right) \left( \fraction{3}{5} \right)^x$$
  4. $$f(x) = 3(8)^x$$
  5. $$h(x) = 0.5(8)^x$$
  6. $$k(x) = 1.3 \left( \fraction{1}{8} \right)^x$$
