Without using a calculator, arrange the following numbers in ascending (increasing) order.

$4\sqrt{3}$, $3\sqrt{5}$, $5\sqrt{2}$, $2\sqrt{10}$, $2\sqrt{13}$, $3\sqrt{6}$

{% capture solution %}

Convert all radicals to entire radicals:

  1. $4\sqrt{3} = \sqrt{48}$
  2. $3\sqrt{5} = \sqrt{45}$
  3. $5\sqrt{2} = \sqrt{50}$
  4. $2\sqrt{10} = \sqrt{40}$
  5. $2\sqrt{13} = \sqrt{52}$
  6. $3\sqrt{6} = \sqrt{54}$

Sort in ascending order: $\sqrt{40}$, $\sqrt{45}$ $\sqrt{48}$, $\sqrt{50}$, $\sqrt{52}$, $\sqrt{54}$

Rewrite sorted order using original radicals: $2\sqrt{10}$, $3\sqrt{5}$, $4\sqrt{3}$, $5\sqrt{2}$, $2\sqrt{13}$, $3\sqrt{6}$
