Simplify the following radicals.

  1. $3\sqrt{x}(2\sqrt{x}+6\sqrt{7})$
  2. $4x\sqrt{x} - 7x\sqrt{x}$
  3. $\displaystyle\frac{18\sqrt{x}}{2\sqrt{x}}$
  4. $\displaystyle\frac{20\sqrt{10x^2}}{5\sqrt{2x}}$
  5. $\displaystyle\frac{10x^2\sqrt{14}}{2x\sqrt{7}}$

{% if true %}
  1. $3\sqrt{x}(2\sqrt{x}+6\sqrt{7}) =$ $6x+18\sqrt{7x}$
  2. $4x\sqrt{x} - 7x\sqrt{x} =$ $-3x\sqrt{x}$
  3. $\displaystyle\frac{18\sqrt{x}}{2\sqrt{x}} = 9$
  4. $\displaystyle\frac{20\sqrt{10x^2}}{5\sqrt{2x}} =$ $4\sqrt{5x}$
  5. $\displaystyle\frac{10x^2\sqrt{14}}{2x\sqrt{7}} =$ $5x\sqrt{2}$
{% endif %}
