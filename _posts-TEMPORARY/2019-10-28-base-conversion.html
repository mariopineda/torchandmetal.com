Convert each of the following powers to the base indicated.

  1. $32^x$ to base 2
  2. $1000^{x+1}$ to base 10
  3. $81^{x-2}$ to base 3
  4. $\left( \displaystyle\frac{25}{49} \right)^{3x}$ to base $\displaystyle\frac{5}{7}$
  5. $\left( \displaystyle\frac{1}{16} \right)^{x+1}$ to base 2.
  6. $\displaystyle\frac{1}{4} \times \left( \displaystyle\frac{1}{16} \right)^{4-x}$ to base 4
{% if true %}
  1. $32^x$ to base 2 $= 2^{5x}$
  2. $1000^{x+1}$ to base 10 $=(10^3)^{x+1}$ $=10^{3(x+1)}$ $10^{3x+3}$
  3. $81^{x-2}$ to base 3 $=(3^4)^{x-2}$ $=3^{4(x-2)}$ $=3^{4x-8}$
  4. $\left( \displaystyle\frac{25}{49} \right)^{3x}$ to base $\displaystyle\frac{5}{7}$ $=\left( \displaystyle\frac{5^5}{7^7} \right)^{3x}$ $=\displaystyle\frac{5^{5 \times 3x}}{t \times 3x}$ $=\displaystyle\frac{5^{15x}}{7^{21x}}$
  5. $\left( \displaystyle\frac{1}{16} \right)^{x+1}$ to base 2 $=2^{-4(x+1)}$ $=2^{-4x-4}$
  6. $\displaystyle\frac{1}{4} \times \left( \displaystyle\frac{1}{16} \right)^{4-x}$ to base 4 $=4^{-1} \times (4^2)^{4-x}$ $=4^{-1} \times 4^{-2(4-x)}$ $=4^{-1} \times 4^{-8+2x}$ $=4^{-1+(-8)+2x}$ $=4^{-9+2x}$
{% endif %}
